# 2주차 단위 테스트 

### 기능요구사항
- [ ] 로또 구입 금액 입력 한다. 
- [ ] 입력한 금액 / 1000 만큼 로또를 구매한다.
- [ ] 수량 만큼 로또를 생성한다.
  - [ ] 로또의 숫자는 1~45의 숫자이다.
  - [ ] 로또는 총 6개의 숫자로 구성되어 있다.
- [ ] 구입한 로또의 수량을 출력한다.
- [ ] 구입한 로또의 번호를 출력한다.
- [ ] 지난 주 당첨 번호를 입력 한다.
- [ ] 3~6개 까지의 일치하는 로또의 수를 구한다.
- [ ] 3~6개 까지의 일치한 로또 수를 출력한다.
- [ ] 수익률을 계산한다. (당첨 금액 / 구입 금액)
- [ ] 총 수익률을 출력한다.
